# Networks-GMC-Backend-
Backend of Social media application Network GMC
# User Routes
* User Signup
* User Login
* User Logout

# Post Routes
* Create Post
* Delete Post By Post Id.
* Edit Post By Id
* Get All Posts
* Get Post By Id

# Like Routes
* Post Like
* Get Like By Id

# Follow Routes
* Follow
* Unfollow

# Comment Routes
* Comment
* Delete Comment
* Edit Comment By Id
* Get Comments By Post ID
* Get Comment By Comment Id
